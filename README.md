# Employee Task Management System

A Java Servlet and JSP based web application for managing employee tasks.

## Features
- User Login System
- Admin Dashboard
- Add Employees
- Assign Tasks
- Employee Task Tracking
- Update Task Status

## Tech Stack
Java  
Servlets  
JSP  
JDBC  
MySQL  
Apache Tomcat  

## Project Structure
src/main/java
   controller
   dao
   dto
   utility

src/main/webapp
   jsp files
   web.xml
